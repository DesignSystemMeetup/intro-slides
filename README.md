Design System Meetup intro slides
===========

> These are the intro slides for the Design System Meetup Sydney.
> https://designsystemmeetup.com/


To see them go to: https://designsystemmeetup.github.io/intro-slides/

-----

To run the slides:

```sh
cd path/to/folder
yarn
yarn build
```

To watch for changes and run a rowsersync server run:

```sh
cd path/to/folder
yarn
yarn watch
```
